Design a **high-end, visually striking AI-powered medical website** that feels like a **premium product experience (Apple-level design + Tesla UI + modern hospital system)**.

The design must **NOT look like a typical AI-generated template**. Avoid repetitive sections, basic grids, or generic card layouts.

Instead, create a **cinematic, layered, and immersive experience** with strong visual storytelling, advanced animations, and interactive medical diagrams.

---

### 🎯 CORE DESIGN DIRECTION (VERY IMPORTANT)

* Use **asymmetrical layouts** (NOT standard grid blocks)
* Combine **large visual sections + floating UI elements**
* Add **depth using layers, blur, transparency, and shadows**
* Create a **scroll-based storytelling experience**
* Each section should feel **unique**, not repeated

---

### 🧠 HERO SECTION (CINEMATIC EXPERIENCE)

* Full-screen immersive hero (NOT a simple banner)

Include:

* **Large 3D human body (center or slightly off-center)**
* Organs (heart, lungs, brain) glowing softly with **realistic animation**
* **Animated scan wave** moving across the body
* Subtle **particle system or light rays** in background

Overlay:

* Floating medical UI panels (like HUD interface)
* Live data (heart rate pulsing, oxygen updating)

Text:

* Bold, large typography (left-aligned or layered over body)
* CTA buttons with smooth hover glow

👉 Make it feel like:
“A live AI system analyzing a human body in real time”

---

### 🧍 INTERACTIVE BODY DIAGRAM SECTION (SIGNATURE EXPERIENCE)

This is what will make your site look **premium and non-AI-generated**.

* Full-width section with **interactive human anatomy**
* On scroll:

  * Different organs highlight (heart → lungs → brain)
* Each organ:

  * Glows
  * Shows tooltip or data card
  * Has animated pulse or movement

Side panels:

* Show:

  * AI diagnosis
  * Risk %
  * Real-time stats

👉 This should feel like a **medical simulation interface**, not a static image

---

### 🤖 AI DASHBOARD (FLOATING, NOT GRID)

Avoid boring dashboard grids ❌

Instead:

* Use **floating panels placed at different depths**
* Overlapping cards with blur backgrounds
* Slight motion (parallax or hover movement)

Include:

* ECG line (smooth animated wave)
* Circular oxygen chart (animated)
* AI insights panel

👉 Make it feel like:
“A futuristic operating system, not a dashboard”

---

### 📊 DATA VISUALIZATION (HIGH-END)

* Use **custom-designed charts**, not default ones

Include:

* Animated line graphs that draw on scroll
* Gradient stroke (blue → cyan glow)
* Floating tooltips that follow cursor

Add:

* Micro-animations (data updating, pulsing points)

---

### 🧬 AI TECHNOLOGY SECTION (VISUAL STORYTELLING)

Instead of cards:

* Use **split layout with visuals + text**
* Left: animated diagram (face scan / body scan / neural network style)
* Right: explanation

Animations:

* Scan lines
* Nodes connecting (AI network effect)
* Subtle glow transitions

---

### 👨‍⚕️ DOCTORS SECTION (PREMIUM, NOT BASIC CARDS)

* Avoid simple grid cards

Use:

* Horizontal scroll or carousel
* Large doctor images with overlay info
* Hover reveals details (experience, specialty)

Add:

* Soft zoom effect
* Gradient overlay on hover

---

### 🏥 SERVICES (NON-GENERIC)

Instead of simple cards:

* Use **icon + description + animated underline**
* Add:

  * Hover transitions
  * Background shift
  * Icon animation

---

### ✨ ANIMATION SYSTEM (THIS IS THE DIFFERENCE)

Use **intentional animations**, not random ones:

* Scroll-triggered reveals
* Parallax layers (foreground vs background)
* Smooth transitions between sections
* Floating UI elements with slow movement
* Organ pulse animation (very subtle, realistic)
* Button hover:

  * Glow + slight scale
* Page should feel alive but **not overloaded**

---

### 🎨 VISUAL STYLE

* White-based professional layout
* Soft gradients (NOT harsh neon)
* Clean spacing (luxury feel)
* Typography:

  * Large headings
  * Minimal but strong hierarchy

---

### 🚫 STRICTLY AVOID

* Basic grid layouts
* Repeated card sections
* Flat UI with no depth
* Overuse of green
* Over-glowing neon effects
* Template-style sections

---

### 💡 FINAL EXPERIENCE GOAL

The website should feel like:

👉 A **live AI medical system**
👉 A **premium SaaS product**
👉 A **real hospital + advanced technology lab**

User reaction should be:

“Wow, this is not a normal website… this feels like a real AI system.”

---

### 🔥 KEY WORDS (IMPORTANT FOR AI TO UNDERSTAND STYLE)

Use these styles:

* cinematic UI
* immersive interface
* layered design
* depth and parallax
* interactive diagrams
* futuristic medical visualization
* premium product design
* non-template layout
